# 기능 스펙: BossSystem

## 개요
플레이어가 준비 완료 시 라스트 보스 도전을 선택하고,
처치 시 승리, 실패 시 패배 처리한다.

---

## 기능 요구사항

### 1. 도전 조건
- 플레이어가 직접 "보스 도전" 버튼을 눌러 시작
- 도전 전 확인 UI 표시 (보스 스탯 정보 + 도전/취소)
- 웨이브 진행 중에는 도전 불가

### 2. 라스트 보스 스탯 (고정)

| 스탯 | 값 |
|------|----|
| HP | 5000 |
| 방어력 | 200 |
| 이동속도 | 2 |
| 플레이어 데미지 | 100 |

- 난이도 단계 공식 미적용 (항상 고정 스탯)

### 3. 보스 동작
- 스폰 포인트에서 등장
- 웨이포인트 경로를 따라 이동
- 기지 도달 시 플레이어 HP 100 감소 → 즉시 패배

### 4. 승패 처리

| 조건 | 결과 |
|------|------|
| 보스 HP 0 이하 | 승리 → Victory 씬 전환 |
| 보스 기지 도달 | 패배 → Defeat 씬 전환 |

### 5. 보스 도전 중 규칙
- 보스 도전 중 일반 웨이브 시작 불가
- 보스 도전 중 큐브 사용/인벤토리 접근 가능
- 보스 도전 실패 시 게임 종료 (재시작 또는 메인 메뉴)
