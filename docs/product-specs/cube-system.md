# 기능 스펙: CubeSystem

## 개요
웨이브 클리어 시 가중치 기반으로 5종 큐브를 드롭하고,
플레이어가 큐브를 소모해 타워 생성, 슬롯 해금, 아이템 크래프팅을 수행한다.

---

## 기능 요구사항

### 1. 큐브 종류 및 용도

| 큐브 | 용도 |
|------|------|
| **하위 큐브** | 타워 생성 / 아이템 옵션 전체 리롤 / 아이템 슬롯 해금 / 상점 구매 |
| **상위 큐브** | 아이템 옵션 1개 추가 / 보조 옵션 슬롯 해금 |
| **최상위 큐브** | 아이템 옵션 1개 제거 → 상위 옵션으로 교체 |
| **삭제 큐브** | 아이템 옵션 1개 랜덤 제거 |
| **복제형 큐브** | 타워 아이템 복제 |

### 2. 큐브 드롭
- 웨이브 클리어 시 가중치 기반 랜덤 드롭
- 실패 시 드롭 없음
- 드롭 가중치 수치: 미확정 (추후 확정)

### 3. 큐브 소모 규칙

| 행동 | 비용 |
|------|------|
| 타워 생성 | 하위 큐브 1개 |
| 아이템 슬롯 1번째 해금 | 하위 큐브 10개 |
| 아이템 슬롯 2번째 해금 | 하위 큐브 20개 |
| 아이템 슬롯 3번째 해금 | 하위 큐브 30개 |
| 보조 옵션 슬롯 1번째 해금 | 상위 큐브 5개 |
| 보조 옵션 슬롯 2번째 해금 | 상위 큐브 10개 |
| 보조 옵션 슬롯 3번째 해금 | 상위 큐브 15개 |
| 상점 스킬/보조옵션 구매 | 하위 큐브 1개 |
| 아이템 옵션 전체 리롤 | 하위 큐브 1개 |
| 아이템 옵션 1개 추가 | 상위 큐브 1개 |
| 아이템 옵션 교체 (상위로) | 최상위 큐브 1개 |
| 아이템 옵션 1개 제거 | 삭제 큐브 1개 |
| 아이템 복제 | 복제형 큐브 1개 |

### 4. 보유량 관리
- 5종 큐브 각각 수량 관리
- 소모 시 보유량 부족하면 실패 처리
- HUD 상단 우측에 실시간 수량 표시

## 인터페이스

```csharp
public class CubeSystem
{
    public int GetCount(CubeType type);
    public bool TryConsume(CubeType type, int amount);
    public void Add(CubeType type, int amount);
    public void DropReward(int stage);    // 웨이브 클리어 시 호출

    public static event Action<CubeType, int> OnCubeChanged;
}

public enum CubeType
{
    Lower, Upper, TopTier, Delete, Clone
}
```
