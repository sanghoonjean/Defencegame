# 기능 스펙: ShopSystem

## 개요
플레이어가 하위 큐브를 소모해 공격 스킬과 보조 옵션을 구매한다.

---

## 기능 요구사항

### 1. 상점 접근
- HUD 하단 우측 상점 버튼으로 열기
- 웨이브 중/외 언제든 접근 가능
- 닫기 버튼으로 종료

### 2. 판매 목록

**공격 스킬 (5종) — 하위 큐브 1개**

| 스킬 | 설명 |
|------|------|
| 단일 레이저 | 단일 대상 고데미지 |
| 미사일 | 범위 폭발 |
| EMP | 스턴/감속 디버프 |
| 레일건 | 직선 관통 |
| 나노봇 | 범위 지속 데미지 |

**보조 옵션 (13종) — 하위 큐브 1개**

| 분류 | 옵션 |
|------|------|
| 공격 강화 | 과부하 모듈, 가속 칩, 광역 증폭기, 다중 투사체, 임계 회로, 치명 증폭기 |
| 상태이상 | EMP 증폭기, 냉각 장치, 부식 탄두, 연소 탄두 |
| 특수 | 연쇄 회로, 관통 탄두, 에너지 흡수 |

### 3. 구매 규칙
- 처음부터 모든 스킬/보조 옵션 구매 가능 (해금 조건 없음)
- 동일 항목 중복 구매 가능 (여러 타워에 장착 가능)
- 보유 하위 큐브 부족 시 구매 불가 (버튼 비활성화)

### 4. 구매 후 처리
- 구매한 스킬/보조 옵션은 인벤토리에 저장
- 타워 선택 후 인벤토리에서 장착

---

## 인터페이스

```csharp
public class ShopSystem
{
    public bool PurchaseSkill(SkillType skill);
    public bool PurchaseSupportOption(SupportOptionType option);
    public List<SkillData> GetOwnedSkills();
    public List<SupportOptionData> GetOwnedSupportOptions();
}
```
