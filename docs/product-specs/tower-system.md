# 기능 스펙: TowerSystem

## 개요
플레이어가 하위 큐브를 소모해 타워를 생성하고,
스킬과 아이템을 장착해 적을 공격한다.

---

## 기능 요구사항

### 1. 타워 생성
- 하위 큐브 1개 소모로 타워 생성
- Buildable 타일에만 배치 가능 (MapTileSystem 검증)
- 생성 시 공격 스킬 없음 — 상점에서 별도 구매 필요

### 2. 타워 구조
```
타워
├── 스킬 시스템
│   ├── 공격 스킬 슬롯: 1개
│   └── 보조 옵션 슬롯: 0~3개 (상위 큐브로 해금)
└── 아이템 시스템
    └── 아이템 슬롯: 0~3개 (하위 큐브로 해금)
```

### 3. 공격 스킬 (5종)

| 스킬 | 공격 타입 | 설명 |
|------|-----------|------|
| 정밀 화살 | 단일 | 단일 대상 고데미지 |
| 화염구 | 범위 | 착탄 지점 범위 폭발 |
| 마비 마법 | 디버프 | 스턴/감속 상태이상 |
| 번개 창 | 관통 | 직선 방향 전체 관통 |
| 독 구름 | 지속 | 일정 범위 지속 데미지 (DOT) |

- 공격 스킬은 자유롭게 교체 가능 (비용 없음)

### 4. 보조 옵션 슬롯 해금

| 슬롯 | 해금 비용 |
|------|-----------|
| 1번째 | 상위 큐브 5개 |
| 2번째 | 상위 큐브 10개 |
| 3번째 | 상위 큐브 15개 |

### 5. 보조 옵션 목록 (13종)

**공격 강화:** 강화 룬, 신속의 룬, 광역 마법진, 다중 화살, 치명 룬, 파괴의 룬

**상태이상:** 마비 룬, 빙결 마법, 부식 독, 화염 마법

**특수:** 연쇄 번개, 관통 화살, 마나 흡수

- 보조 옵션은 자유롭게 교체 가능 (비용 없음)

### 6. 공격 동작
- 공격 범위 내 적 감지 시 자동 공격
- 장착된 스킬 타입에 따라 공격 방식 결정
- 데미지 계산: `실제 데미지 = (기본 공격력 + 아이템 옵션 합산) - 적 방어력` (최소 1)

### 7. 타워 선택
- 타워 클릭 시 인벤토리/스킬 패널 팝업
- 선택된 타워의 스킬, 보조 옵션, 아이템 슬롯 표시

## 인터페이스

```csharp
public class Tower
{
    public SkillData EquippedSkill { get; }
    public List<SupportOptionData> SupportOptions { get; }
    public List<ItemData> Items { get; }

    public void EquipSkill(SkillData skill);
    public void AddSupportOption(SupportOptionData option, int slot);
    public bool UnlockSupportSlot(int slot);   // 상위 큐브 소모
    public void RefreshStats();

    public static event Action<Tower> OnTowerPlaced;
}
```

## 검증 조건
- [ ] 하위 큐브 1개 소모 후 타워 생성
- [ ] Buildable 타일 외 배치 불가
- [ ] 스킬 교체 시 비용 없음
- [ ] 보조 옵션 슬롯 해금 비용 정확히 소모
- [ ] 공격 범위 내 적 자동 감지 및 공격
- [ ] 데미지 계산 공식 정확히 적용
