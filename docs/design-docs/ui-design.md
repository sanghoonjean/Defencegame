## 3. 씬별 UI 상세

### 메인 메뉴
- 게임 타이틀 (MakeDefence)
- 게임 시작 버튼
- 설정 버튼
- 종료 버튼

### 게임 화면
- 맵 + 타워 배치
- HUD (2번 항목 참조)
- 타워 클릭 시 → 인벤토리/스킬 패널 팝업

### 웨이브 결과
- 클리어/실패 여부
- 획득한 큐브 목록 및 수량
- 다음 웨이브 진행 버튼

### 상점
- 공격 스킬 목록 (5종) + 가격 (하위 큐브 1개)
- 보조 옵션 목록 (13종) + 가격 (하위 큐브 1개)
- 보유 큐브 수량 표시
- 닫기 버튼

### 인벤토리
- 타워 아이템 슬롯 3개 표시
- 슬롯별 해금 상태 / 해금 버튼 (하위 큐브 10/20/30개)
- 아이템 옵션 목록 표시
- 큐브 사용 버튼 (하위/상위/최상위/삭제/복제형)
- 스킬 슬롯 표시 (공격 스킬 + 보조 옵션 슬롯 0~3개)
- 보조 옵션 슬롯 해금 버튼 (상위 큐브 5/10/15개)

### 보스 도전
- 보스 스탯 정보 표시
- 도전 확인 / 취소 버튼

### 승리 / 패배
- 결과 텍스트
- 메인 메뉴로 버튼
- 재시작 버튼

---

## 4. UI 설계 원칙

- 큐브 수량, 옵션 확률 등 모든 수치는 플레이어에게 투명하게 공개 (`core-beliefs.md` 5원칙)
- 웨이브 중에도 인벤토리/큐브 사용 가능
- 타워 선택 시 해당 타워 정보 즉시 확인 가능
