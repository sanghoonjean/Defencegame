# Tower Design — MakeDefence

## 1. 타워 기본 구조

타워는 1종류이며, 스킬 슬롯과 아이템 슬롯 구성에 따라 역할이 결정된다.

```
타워
├── 스킬 시스템
│   ├── 공격 스킬 슬롯: 1개 (5종 중 택1)
│   └── 보조 옵션 슬롯: 최대 3개 (상위 큐브로 해금)
│
└── 아이템 시스템
    └── 아이템 슬롯: 3개 (큐브로 옵션 크래프팅)
```

---

## 2. 공격 스킬 5종

| 스킬 | 공격 타입 | 설명 |
|------|-----------|------|
| **단일 레이저** | 단일 | 단일 대상 고데미지 |
| **미사일** | 범위 | 착탄 지점 범위 폭발 |
| **EMP** | 디버프 | 스턴/감속 상태이상 부여 |
| **레일건** | 관통 | 직선 방향 전체 관통 |
| **나노봇** | 지속 | 일정 범위 내 지속 데미지 (DOT) |

- 스킬은 상점에서 **하위 큐브 1개**로 구매
- 장착된 스킬은 **자유롭게 교체** 가능 (비용 없음)
- 처음부터 모든 스킬 구매 가능

---

## 3. 보조 옵션 슬롯 해금

보조 옵션 슬롯은 기본 0개이며, 상위 큐브를 소모해 순서대로 해금한다.

| 슬롯 | 해금 비용 |
|------|-----------|
| 1번째 슬롯 | 상위 큐브 5개 |
| 2번째 슬롯 | 상위 큐브 10개 |
| 3번째 슬롯 | 상위 큐브 15개 |

---

## 4. 보조 옵션 목록 (13종)

보조 옵션은 상점에서 **하위 큐브 1개**로 구매. 장착/교체는 자유롭게 가능.

### 공격 강화

| 보조 옵션 | 효과 |
|-----------|------|
| 과부하 모듈 | 공격력 증가 |
| 가속 칩 | 공격 속도 증가 |
| 광역 증폭기 | 공격 범위 증가 |
| 다중 투사체 | 투사체 수 증가 |
| 임계 회로 | 치명타 확률 증가 |
| 치명 증폭기 | 치명타 데미지 증가 |

### 상태이상

| 보조 옵션 | 효과 |
|-----------|------|
| EMP 증폭기 | 기절 확률 증가 |
| 냉각 장치 | 적 이동속도 감소 확률 |
| 부식 탄두 | 적 방어력 감소 확률 |
| 연소 탄두 | 적중 시 지속 데미지 (DOT) 추가 |

### 특수

| 보조 옵션 | 효과 |
|-----------|------|
| 연쇄 회로 | 공격이 인접 적에게 연쇄 |
| 관통 탄두 | 투사체가 적을 관통 |
| 에너지 흡수 | 적 처치 시 하위 큐브 드롭 확률 증가 |

---

## 5. 타워 생성

- **하위 큐브 1개** 소모로 타워 생성
- 생성 시 기본 공격 스킬 없음 — 상점에서 별도 구매 필요
- 배치는 MapTileSystem에서 허용된 타일에만 가능

---

## 6. 미확정 항목

- [ ] 공격 스킬별 기본 수치 (데미지, 범위, 쿨타임 등)
- [ ] 보조 옵션별 수치 (증가량 %)
- [ ] 타워 기본 스탯 (체력, 기본 공격력 등)
- [ ] 추가 보조 옵션 (추후 확장)
- [ ] 속성 데미지 시스템 — 공격 스킬/보조 옵션에 속성(화염, 냉기, 전기 등) 부여 가능성 검토
