# Map Design — MakeDefence

## 1. 맵 기본 정보

| 항목 | 내용 |
|------|------|
| 맵 종류 | 단일 고정 맵 |
| 맵 크기 | 60x33 타일 (1920x1080 해상도 기준, PPU 32) |
| 카메라 | 고정 (스크롤 없음) |
| 경로 방식 | 웨이포인트 (단일 고정 경로) |

---

## 2. 맵 구조

3x3 구역 그리드로 구성. 초록 경로(길)가 구역 사이를 연결하며 흐른다.

```
[스폰]     [본진]
  ↓          ↑
┌──────┬──────┬──────┐
│ 초록 │ 파랑 │ 초록 │  ← 상단 행
├──────┼──────┼──────┤
│ 파랑 │ 파랑 │ 파랑 │  ← 중간 행
├──────┼──────┼──────┤
│ 초록 │ 파랑 │ 초록 │  ← 하단 행
└──────┴──────┴──────┘
```

| 구역 색상 | 의미 |
|-----------|------|
| **초록 테두리** | 경로 인접 구역 — 타워 설치 가능 |
| **파란 테두리** | 경로 비인접 구역 — 타워 설치 불가 |

---

## 3. 웨이포인트 경로

스폰(좌상단) → 본진(중상단) 방향으로 이동.

```
[스폰] ──→ 하단 이동
           ──→ 우측 이동
                ──→ 상단 이동
                     ──→ 좌측 이동
                          ──→ [본진] 도달
```

경로는 3x3 구역 테두리(초록선)를 따라 흐르며,
적은 웨이포인트 순서대로 이동한다.

---

## 4. 타일 종류

| 타일 | 설명 | 타워 배치 |
|------|------|-----------|
| **경로 타일** | 적이 이동하는 길 (초록선) | 불가 |
| **배치 가능 타일** | 초록 테두리 구역 내부 | 가능 |
| **배치 불가 타일** | 파란 테두리 구역 내부 | 불가 |
| **장식 타일** | 배경, 구조물 등 시각 요소 | 불가 |

---

## 5. 주요 지점

| 지점 | 위치 | 설명 |
|------|------|------|
| **스폰 포인트** | 좌상단 | 웨이브 적 등장 지점 |
| **본진 (기지)** | 중상단 | 적 도달 시 플레이어 HP 감소 |

---
