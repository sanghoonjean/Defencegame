## 2. 맵 구조

3x3 구역 그리드로 구성. 경로(길)는 구역 사이 통로를 따라 흐른다.
**모든 9개 구역이 타워 설치 가능 구역**이며, 통로(경로)에는 설치 불가.

```
[스폰]   [본진]
  │    ←←←←←←←←←←←←←←│
  ↓  ┌──────┬──────┬──────┤
     │  ㅁㅁ │  ㅁㅁ │  ㅁㅁ │
     │  ㅁㅁ │  ㅁㅁ │  ㅁㅁ │
  ↓  ├──────┼──────┼──────┤  ↑
     │  ㅁㅁ │  ㅁㅁ │  ㅁㅁ │
     │  ㅁㅁ │  ㅁㅁ │  ㅁㅁ │
  ↓  ├──────┼──────┼──────┤  ↑
     │  ㅁㅁ │  ㅁㅁ │  ㅁㅁ │
     │  ㅁㅁ │  ㅁㅁ │  ㅁㅁ │
  └→→→→→→→→→→→→→→→→→→→→┘
```

## 3. 웨이포인트 경로

스폰(좌상단) → 본진(중상단) 방향으로 이동.

```
[스폰]
  ↓ 좌측 통로 하강
  → 하단 통로 우측 이동
  ↑ 우측 통로 상승
  ← 상단 통로 좌측 이동
  → [본진] 도달
```

적은 웨이포인트 순서대로 구역 사이 통로를 이동한다.

## 4. 타일 종류

| 타일 | 설명 | 타워 배치 |
|------|------|-----------|
| **경로 타일** | 구역 사이 통로, 적이 이동하는 길 | 불가 |
| **배치 가능 타일** | 9개 구역 내부 (ㅁ) | 가능 |
| **장식 타일** | 배경, 구조물 등 시각 요소 | 불가 |

---

## 5. 주요 지점

| 지점 | 위치 | 설명 |
|------|------|------|
| **스폰 포인트** | 좌상단 | 웨이브 적 등장 지점 |
| **본진 (기지)** | 중상단 | 적 도달 시 플레이어 HP 감소 |
