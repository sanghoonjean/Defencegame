# 스킬 구현 계획: Phase 9 — 이동 스킬 (Movement Skills)

## 목표
이동 계열 스킬 4종을 구현한다.
타워 디펜스 맥락에서 이동 스킬은 주로 **적의 특수 이동 패턴** 또는
**플레이어/타워 위치 변경** 메커니즘에 활용한다.

## 선행 조건
- EnemySystem 이동 패턴 확장 가능
- Phase 1~8 스킬 시스템 완료

## 타워 디펜스 적용 방향

이동 스킬은 플레이어 캐릭터가 없는 타워 디펜스 장르에서
아래 두 가지 방향 중 하나로 적용한다.

**A. 적 특수 패턴 스킬로 활용**
- Unique/LastBoss 등급 적이 사용하는 특수 이동 능력으로 구현
- 정해진 웨이포인트 경로를 벗어나 타워를 공격하거나 기지로 돌진

**B. 타워 특수 스킬로 활용**
- 배치된 타워가 일정 쿨타임마다 발동하는 이동형 공격
- 타워가 고정 위치에서 이동 투사체를 발사하는 형태로 추상화

---

## 스킬별 구현 작업

### 1. 도약 강타 (Leap Slam)
**적 패턴 활용:**
- [ ] Unique 등급 적이 원거리 타워를 향해 도약 후 착지 충격 발생
- [ ] 착지 지점 AoE 폭발, 주변 타워 피해

**타워 스킬 활용:**
- [ ] 타워가 인접 범위 내 적에게 도약 공격 발사 (애니메이션 투사체)
- [ ] 착지 지점 물리 AoE

### 2. 방패 돌진 (Shield Charge)
**적 패턴 활용:**
- [ ] 돌진형 적이 직선 경로로 빠르게 이동하며 경로 상 타워 피해
- [ ] 돌진 후 충격 AoE

**타워 스킬 활용:**
- [ ] 타워 전방 직선으로 돌진형 투사체 발사
- [ ] 경로 상 적 다수 관통 타격

### 3. 칼날 선회 (Whirling Blades)
**적 패턴 활용:**
- [ ] 근접 적이 회전하며 이동, 경로 상 타워 피해
- [ ] 이동 중 범위 내 모든 아군에 소량 지속 피해

**타워 스킬 활용:**
- [ ] 타워에서 회전형 투사체 발사, 경로 상 적 반복 타격
- [ ] 이동 거리와 피해 횟수 설정

### 4. 점멸 타격 (Flicker Strike)
**적 패턴 활용:**
- [ ] Unique 등급 이상 적이 랜덤 타워로 순간 이동 후 타격
- [ ] 텔레포트 이펙트 후 해당 타워 피해

**타워 스킬 활용:**
- [ ] 타워가 쿨타임마다 가장 강한 적 위치로 순간 이동 공격 발사
- [ ] 즉발 고데미지 단일 타격

## 공통 컴포넌트

### EnemyMovementSpecial
- [ ] 기존 웨이포인트 이동을 일시 중단하고 특수 이동 패턴 실행
- [ ] 특수 이동 완료 후 웨이포인트 경로 재개

### TeleportAttack
- [ ] 목표 위치로 즉발 이동 후 데미지 적용
- [ ] 이동 이펙트(VFX) 연계
